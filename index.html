CSS Transitions

CSS transitions allow property changes to occur smoothly over a specified duration.

Hover over the red square to see a transition effect.

.animated-element {
transition: all 0.4s ease;
}

.animated-element:hover {
transform: rotate(45deg) scale(1.2);
background: #4ecdc4;
border-radius: 50%;
}

Keyframe Animations

CSS keyframes allow you to create complex animations with multiple steps.

This blue circle uses a keyframe animation to pulse continuously.

@keyframes pulse {
0% { transform: scale(1); opacity: 1; }
50% { transform: scale(1.2); opacity: 0.7; }
100% { transform: scale(1); opacity: 1; }
}

.keyframe-demo {
animation: pulse 2s infinite;
}
